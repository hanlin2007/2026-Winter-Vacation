#### （一）C++和 C 的关系，从C到C++的演进

现代很多编译器可以同时编译.c文件和.cpp文件，C++对C是完全向下兼容的。

C语言：面向过程，核心是函数和数据结构，程序由函数调用构成，数据和操作分离。
C++：增加了面向对象和泛型编程特征。

什么是**数据和操作分离**：数据结构如struct只包含成员变量，没有成员函数。操作这些数据的函数需要独立定义，需要显示将数据作为参数进行传递。

**面向对象编程**：将数据和对数据的操作封装在一起，通过类与对象来组织代码，强调**抽象、封装、继承、多态**
封装：将数据和方法放在一个类中，通过访问修饰符来控制访问权限
继承：子类继承父类的属性和方法，可以进行扩展、重写
多态：同一方法接口，如Shape基类中的draw() 方法，Circle和Rectangle有各自的实现
抽象：提取共性，定义接口

**泛型编程**：编写与类型无关的算法和数据结构，使用模板（template）实现

```
template <typename T>
T max(T a,T b){
    return (a>b)?a:b;
}
```

容器类：vector\<T>  list\<T>
通用算法：sort( ) , find( )

**STL**是什么：Standard Template Library，标准模板库，是C++的一个标准库，基于**泛型编程**实现，包含三大核心组件：
容器（Containers）用于存储数据，如vector，list，map，set
算法（Algorithms）对容器进行操作的函数，如sort，copy，find
迭代器（Iterators）连接容器和算法，类似指针，用于遍历容器元素
